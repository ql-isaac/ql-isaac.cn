Codespaces

Codespaces 是 GitHub 推出的在线编辑器，简单点说，就是可以实现将你本地的以 VsCode 为核心的开发环境迁移到云端，这样你只需要使用浏览器就可以随时随地进行开发，还可以多人协作。

目前这一云服务还不能免费使用，但是有一部分功能可以免费使用，那就是升级版的代码浏览编辑功能，请看下图。
